# 🚀 Automation Framework using Selenium + Cucumber BDD

## 📌 Project Overview
This project demonstrates an end-to-end test automation framework built using Selenium WebDriver with Java and Cucumber (BDD approach).

The framework is designed using Page Object Model (POM) to ensure scalability, maintainability, and reusability of test scripts. It simulates real-world user scenarios such as login, search, and cart functionality.

---

## 🛠️ Tech Stack
- Selenium WebDriver
- Java
- Cucumber (BDD)
- TestNG
- Maven
- Git
