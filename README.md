# Automation Using Cucumber BDD

## 📌 Project Overview
This project is an automation testing framework built using Cucumber BDD, Selenium WebDriver, and Java.  
It follows Behavior Driven Development (BDD) principles where test scenarios are written in Gherkin language.

The framework helps in creating readable, maintainable, and scalable test automation scripts.
